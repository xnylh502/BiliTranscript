# B站视频文案提取器（Python 桌面原型）

> 这是项目早期的桌面网页版原型。主力已转向 [安卓端](../README.md)，此版仅作参考。

输入 B站 视频链接，AI 自动识别语音并转为文字文案。

## 特性

- 卡牌风格 Web 界面，一键提取
- 支持 B站 分享文本自动解析（`【标题】 https://...`）
- **双引擎识别**：本地 Whisper + 百度语音 API（云端）
- 配置百度 API 后自动切换云端识别
- 文案自动合并为自然段落
- 支持复制文案、下载 TXT 文件

## 使用方法

Windows 用户直接双击 `start.bat`：自动安装依赖 → 自动打开浏览器 `http://127.0.0.1:5000`，粘贴 B站 分享链接即可提取文案。

## 双引擎说明

### 引擎 1：本地 Whisper（默认，无需配置）
- 模型文件放在 `models/` 目录（small 模型，约 461MB；该文件未入库，需自行准备）
- 无需联网，开箱即用
- 识别速度：1 分钟视频约 9 秒；中文准确率约 85–90%

### 引擎 2：百度语音 API（可选，准确率更高）
- 云端识别，速度更快，中文约 95%+
- 免费额度：个人认证可领 5 万次调用 + 10 小时音频转写
- 申请后把三个密钥（AppID / API Key / Secret Key）按行填入项目根目录的 `baidu_key.txt`（该文件已被 `.gitignore` 排除，切勿提交）

## 文件说明

| 文件 | 说明 |
|------|------|
| `start.bat` | 一键启动脚本 |
| `launcher.py` | 启动器（环境检测） |
| `app.py` | Flask Web 服务 |
| `audio_extractor.py` | 核心：下载音频 + 双引擎识别 |
| `cloud_asr.py` | 百度语音 API 封装 |
| `utils.py` | 链接解析工具 |
| `bilibili_api.py` | B站 字幕抓取 |
| `templates/index.html` | 前端页面 |

## 注意事项
